Небольшое приложение, с помощью которого можно получить свой консольный переводчик, или же имплементировать в ваше приложение.
В main.cpp приведён пример использования данных файлов.
Из-за примененного паттерна проектирования фасад - можно легко расширять функционал приложения, например подключить другую API, или же сделать декоратор, который будет работать с переводом целых файлов.
В EncryptionDecorator приведён лишь пример того, как может работать подобный класс, на деле можно дописать реализацию шифрования файла
